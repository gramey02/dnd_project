# s_het Data Repository

S_het provides a proxy measurement for haplosufficiency in terms of the magnitude of selection against heterozygous loss-of-function mutations. S_het data was downloaded from https://zenodo.org/records/10403680, which comes from the publication:
Zeng, T., Spence, J. P., Mostafavi, H. & Pritchard, J. K. s_het estimates from GeneBayes and other supplementary datasets. Zenodo https://doi.org/10.5281/zenodo.10403680 (2023).
