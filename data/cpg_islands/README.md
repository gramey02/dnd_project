# CpG Island Data Directory

Data was donwloaded from the UCSC Genome Browser by navigating to the Table Browser (https://genome.ucsc.edu/cgi-bin/hgTables?hgsid=3853014969_T6SxPEaTpTPiAsWyR9thdaizGGQd) and selecting "Regulation" as the _Group_, "CpG Islands" as the _Table_, and "Genome" in _Define Region of Interest_.

Headers were removed for ease of use in scripts, to generate CpG_islands_noHeader.txt file.
